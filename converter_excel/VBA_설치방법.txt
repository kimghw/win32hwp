============================================================
엑셀 VBA 매크로 설치 방법
============================================================

[목적]
- 엑셀에서 버튼 클릭으로 한글 문서에 데이터 전송
- 한글 문서에서 엑셀로 데이터 가져오기

[기능]
- SendToHwp: 엑셀 → 한글 (배경색 없는 셀만)
- GetFromHwp: 한글 → 엑셀 (배경색 없는 셀만)
- CreateButtons: 시트에 버튼 자동 생성

============================================================
설치 순서
============================================================

1. 엑셀 파일을 .xlsm 으로 저장
   - 파일 → 다른 이름으로 저장
   - 파일 형식: "Excel 매크로 사용 통합 문서 (*.xlsm)"

2. VBA 편집기 열기
   - Alt + F11

3. 모듈 추가
   - 삽입 → 모듈

4. vba_macro.bas 내용 붙여넣기
   - vba_macro.bas 파일을 메모장으로 열기
   - 전체 복사 (Ctrl+A → Ctrl+C)
   - VBA 편집기의 모듈에 붙여넣기 (Ctrl+V)

5. 참조 추가 (선택사항 - 오류 시)
   - VBA 편집기 → 도구 → 참조
   - "HWP Type Library" 체크
   - 없으면: 찾아보기 → C:\Program Files (x86)\HNC\Hwp80\hwpobject.tlb

6. 버튼 생성
   - Alt + F8 → CreateButtons 실행
   - 또는 개발 도구 탭에서 직접 버튼 추가

7. 저장 후 사용
   - 매크로 보안 경고 시 "콘텐츠 사용" 클릭

============================================================
사용 방법
============================================================

[한글로 보내기]
1. 한글 프로그램에서 해당 문서 열기
2. 엑셀에서 "한글로 보내기" 버튼 클릭
3. 배경색 없는 셀의 값이 한글로 전송됨

[한글에서 가져오기]
1. 한글 프로그램에서 해당 문서 열기
2. 엑셀에서 "한글에서 가져오기" 버튼 클릭
3. 한글 셀의 값이 엑셀로 복사됨

============================================================
주의사항
============================================================

- 반드시 한글 프로그램이 실행 중이어야 함
- 한글 문서가 열려 있어야 함
- 배경색 있는 셀은 편집 불가 (잠금 상태)
- _cells 시트의 list_id 정보가 필요함

============================================================
문제 해결
============================================================

Q: "한글이 실행 중이지 않습니다" 오류
A: 한글 프로그램을 먼저 실행하세요

Q: "자동화 오류" 발생
A: 한글 보안 모듈 등록 필요
   - 한글에서 도구 → 스크립트 매크로 → 보안 설정

Q: 값이 전송되지 않음
A: _cells 시트의 list_id가 올바른지 확인
   - list_id는 한글 문서 구조에 따라 달라짐
   - 문서가 변경되면 다시 export 필요
